COMANDOS GIT

git status  : Proporciona información sobre el estado actual de un repositorio Git y su contenido.
git log		: Es todo el historial
git init	: Crea un repositorio de Git
git add	fichero	: Prepar los cambios para ser validados
git add *	: Añade todo lo que esté modificado al staging
git rm --cached <file> : sacarlo del staging
gt commit -m "mensaje informativo"	: Valida y guarda cambios en el repositorio local
git push	: Sube los cambios al repositorio remoto y crea una nueva versión
git clone	: Clona/copia un repositorio remoto en una versión local
git pull	: Descarga los cambios del repositorio remoto
git branch name : Crear una rama
git branch      : muestra en que rama estoy
git checkout branch_name : Saltar de una rama a otra
git switch :  ¡cuidado porque hace más cosas!
